# Math formulas document

## Requirement with inline math

**UID**: REQ-MATH-1

The spacecraft dry mass is $m_d = 120\,kg$ and the propellant mass is $m_p = 30\,kg$.

The total mass shall satisfy $m_t = m_d + m_p \le 160\,kg$.
